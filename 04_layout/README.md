##Lesson 04 - Layout


###LEARNING OBJECTIVES


*	Draw the DOM Tree for web pages containing  ```<div>``` & ```<section>```.

*	Apply header, footer, sidebar, and multi-column layouts to develop a web page.

*	Experiment and predict effects of floats and clearing CSS positioning.
